    Лабораторная работа 3

    НАСТРОЙКА ЛОКАЛЬНОЙ СЕТИ ПЕРЕДАЧИ ДАННЫХ

    Настройка коммутаторов и маршрутизаторов

    Задачи:

    • Соединить коммутаторы между собой и проверить работоспособность сети
    • Настроить интерфейсы маршрутизатора для существующих подсетей
    • Соединить маршрутизаторы и настроить между ними сеть
    • Настроить статическую маршрутизацию между сетями

    Данная лабораторная работа представляет собой настройку небольших сетей. Имеется три сети, в каджой из которых по одному коммутатору и два компьютера. Кадждой сети соответствует свой маршрутизатор. Для всех сетей задан свой пул адресов с определенными масками сети в соответствии с вариантом. Для всех устройств были установлены IP-адреса из заданного пула. Была настрона статическая маршрутизация к одной из сетей. При добавлении в рабочее пространство двух коммутаторов с их компьютерами и соединении этих коммутаторов кабелем, пакеты отправленные из одной сети не могли попасть в другую, поскольку для каждого из коммутаторов были настроены свои адреса и маски. Для обеспечения взяимодействия двух сетей используют маршрутизаторы.
